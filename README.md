# Calendrier

##DESCRIPTION

A simple helper for creating an HTML calendar. 
It allows create events too.

##SYNOPSIS:


    # Simple
    <%= calendrier(@events, :year => 2012, :month => 4) %>  


## INSTALLATION

Add this line to your application's Gemfile:

    gem 'calendrier'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install calendrier
